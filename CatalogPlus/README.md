CatalogPlus v. 2.0
=========
###Описание
Данный набор модулей предназначен для отображение категорий и товаров на сайте без возможности их складывания в корзину(для этого есть отдельный модуль).

###Требования
1. Обязательно подключать в конфиге CatalogPlus_Catagory, CatalogPlus_Good
2. Обязательно наличие Medium CatalogList. Он нужен для создание связей товара и категорий
3. Данная версия писалась под IdealCMS v2.1(с addon)
4. В админке нужно делать вот такую структуру страниц, так что бы **модули CatalogPlus находились на одном уровне**. 
***Пример:***

* **Гланая**
* Part - 1
* Part - 2
* ...
* Ideal_Part(Обычная страница в админке) - **Для модуля CatalogPlus**
 1. Part - тип документа ***CatalogPlus_Good(Товары)***
 1. Part - тип документа ***CatalogPlus_Category(Категории)***
* ...
* Part -последняя

###Таблицы
1. prefix **catalogplus_structure_category** - тут хранятся категории
2. prefix **catalogplus_structure_good**     - тут хранятся товары
3. prefix **catalogplus_medium_cataloglist** - тут хранятся зависимости товара к категориям

###Остальное
В версии 1.2 изменена систем вызова контроллеров.
Теперь для категорий товара всегда вызывается контроллер категорий,
а контроллер товара срабатывает только на карточке товара.

Внутри модели CatalogPlus_Category всегда находится модель
CatalogPlus_Good, а в модели CatalogPlus_Good всегда находится
модель CatalogPlus_Category.
